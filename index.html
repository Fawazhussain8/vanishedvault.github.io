Overview

Welcome, digital detectives! In the Vanished Vault challenge, your mission is to unravel hidden clues through document analysis, website archival investigations, and image steganography.

Challenge Instructions

  1. Document Analysis: Examine the blueprint image’s metadata to find a hidden URL. The image is password protected; the key lies in the accompanying archaeologist’s image.
  2. Website Archive Investigation: Use the URL from the metadata to locate an archived webpage. Inspect its source for a hidden clue – a secret directory.
  3. Image Analysis: Navigate to the secret directory to retrieve an image. Deduce the location from visual clues and use it as the passphrase for steganographic extraction of the final flag.

Hints

  • Run exiftool on the blueprint image.
  • Check the webpage’s HTML source for hidden comments.
  • Use tools like steghide to extract data from images.

Good luck – the vault awaits your discovery!
